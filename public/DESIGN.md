## Design Decisions

* Font Family: Poppins
* Designed four website pages within a single Figma file
* Created both Dark Mode and Light Mode versions of each page
* Focused on a minimal and clean design approach
* Primary color palette: Black, Grey, and White
* Prioritized readability, responsiveness, and user experience

## Pages Designed

* Home Page
* Projects Page
* Journey Page
* Contact Page

## Goal

The primary goal of the design was to create a modern, professional, and easy-to-navigate portfolio website while maintaining visual consistency across all pages.
